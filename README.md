# Prutal

Prutal is a pure Go alternative to [protocol buffers](https://protobuf.dev), it covers most of the functionality offered by Protocol Buffers.

Prutal aims to minimize code generation as much as possible while ensuring serialization and maintaining good performance.

**Since Prutal is NOT yet ready for production use, we are not providing usage documentation at this time, nor do we guarantee backward compatibility of the interface.**

## Protobuf Compatibility

* вң… Works with code generated by the official Protocol Buffer Go

### Core Message Operations

* вң… Marshal / Unmarshal вҖ” binary wire format
* вң… MarshalAppend вҖ” append-friendly encoding
* вң… Size вҖ” compute wire size without marshaling (zero allocation)
* вң… Reset
* вқҢ Clone / Merge / Equal
* вқҢ CheckInitialized вҖ” validate required field presence
* вқҢ MarshalOptions / UnmarshalOptions (deterministic output, discard unknown, recursion limits, etc.)

### Proto Language Support

* вң… Scalar types: all 15 protobuf scalar types
* вң… Messages and nested messages
* вң… Repeated fields
* вң… Enums (including `allow_alias`)
* вң… Optional / field presence
* вң… Map fields
* вң… Oneof fields
* вң… Required fields (proto2)
* вң… Packed / unpacked encoding
    - PACKED / EXPANDED (repeated field encoding, edition 2023)
* вң… Unknown field preservation
* вң… Reserved fields and names
* вқҢ [Default scalar values (proto2, edition 2023)](https://protobuf.dev/programming-guides/proto2/#explicit-default)
* вқҢ [Groups (proto2, deprecated)](https://protobuf.dev/programming-guides/proto2/#groups)
* вқҢ UTF-8 validation
* вқҢ Message sets (legacy format)

### Encoding Formats

* вң… Binary wire format
* вҡ пёҸ JSON: struct tag only, no `protojson` equivalent
* вқҢ Text format (`prototext`)
* вқҢ Delimited format (length-prefixed message streams)

### Reflection

* вқҢ Descriptor-based reflection (`protoreflect`)
* вқҢ Type registries (`protoregistry`)
* вқҢ Dynamic messages (`dynamicpb`)

### Extension Support

* вҡ пёҸ Extensions (proto2): parsed but limited codegen
* вқҢ Has / Get / Set / Clear / Range extensions

### Code Generation (`prutalgen`)

* вң… Proto2, proto3, edition 2023
* вң… Imports (regular, public, weak)
* вң… gRPC / Services (client/server stubs, streaming)
* вҡ пёҸ Go only
* вҡ пёҸ [Well-known types](https://protobuf.dev/reference/protobuf/google.protobuf/): reuses [`google.golang.org/protobuf/types/known`](https://pkg.go.dev/google.golang.org/protobuf/types/known), `Any` is limited
* вҡ пёҸ Custom options: subset of gogoproto options
* вқҢ Edition 2024
* вқҢ Opaque API / lazy decoding
* вқҢ Deprecation markers
